How it works

The tag carries a short, stable slug URL. Where that slug goes lives in editable config — so you reprogram the tag without ever rewriting it.

01

Write a slug

Put neves.cloud/tapto/#desk on an NFC tag or print it as a QR. Once.

02

Point it anywhere

A link, a phone number, Wi-Fi details, a note rendered in place — any scheme.

03

Change it anytime

Edit the destination from the creator. The tag in the wild stays exactly the same.

Reprogrammable

Repoint a tag by editing config, never by re-writing the tag. Print once, change forever.

Any carrier

NFC, QR, or a plain link — the same slug works everywhere. A tapto QR is dynamic, not baked.

Yours

Runs on your own domain, your data in your GitHub. No middleman, no lock-in.
